All the websites / other things listed don’t cost anything to use. They may include subscription options, but most, if not all, main features are available to everyone.

Learn a language

Duolingo: https://www.duolingo.com/
Gamified learning for any language you can think of

Memrise: https://www.memrise.com/
Same concept as Duolingo, focuses a lot more on vocabulary

LingoDeer (mobile only): https://www.lingodeer.com/
Better learning for Asian languages, very similar to Duolingo

Lingvist: https://lingvist.com/
In case you need more vocab practice

Tinycards: https://tinycards.duolingo.com/
Flashcards to help brush up on concepts and vocab from Duolingo courses

Learn coding

Automate the Boring Stuff with Python: https://automatetheboringstuff.com/
Free to read book on Python, teaching you how to do practical things with it

Codecademy: https://www.codecademy.com/
Courses for a bunch of programming concepts and languages

repl.it: https://repl.it/
Web based programming environment for most programming languages

Codewars: www.codewars.com/r/ai0z2A
For advanced coders, practice your skills here
